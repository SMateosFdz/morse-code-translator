
# Morse code translator with arduino

Simple program to translate words, numbers and phrases to morse code.


## Features

- Simple code
- Comments in english
- Translation of uppercase and lowercase letters


## Documentation

Book used to create this program: [Programming Arduino: Getting Started with Sketches, Second Edition](https://www.amazon.com/Programming-Arduino-Getting-Started-Sketches/dp/1259641635)
